## webpack-to-vite
convert a webpack project to a vite project

## Quick Start

1. download
```
git clone https://github.com/originjs/webpack-to-vite.git
cd webpack-to-vite
```
2. install

with npm, run
```
npm install
npm run build
```
with yarn, run
```
yarn
yarn build
```
3. convert
```
node ./bin/index --rootDir <project path>
```
